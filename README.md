# iClipClap auto-update artifacts

Public repository hosting build artifacts and the Sparkle appcast feed
for [iClipClap](https://github.com/vdzerkalo/iClipClap) (private source).

Contents are overwritten on each tag push by the source repo's release
workflow. Don't open issues here — file them on the source repo.

License: artifacts are MIT, same as iClipClap.
